Active Directory Help Desk Lab

* Overview
This project is a hands-on simulation of a real IT help desk environment using Active Directory and the osTicket ticketing system. It focuses on common day to day support tasks such as password resets, account unlocks, and new employee onboarding.

* Environment Setup
- Windows Server ( AWS EC2 )
- Active Directory Domain Services ( AD DS )
- osTicket Ticketing System
- Remote Desktop (RDP)

*Domain:* lab.local
---

* Project 1: Password Reset
A user was unable to log in due to incorrect credentials.

**Actions Taken:**
- Located the user in Active Directory
- Reset the password
- Verified the user was able to log in successfully

---

* Project 2: Account Lockout
A user’s account was locked after multiple failed login attempts.

**Actions Taken:**
- Verified the account was locked
- Unlocked the account in Active Directory
- Confirmed the user could log in

---

 *  Project 3: New Employee Onboarding
HR requested a new account for a new employee.

**Actions Taken:**
- Created a new user in Active Directory
- Added the user to the HR-Team security group
- Provided login credentials
- Verified access to the system

---

- Skills Demonstrated
- Active Directory user and account management
- Troubleshooting login and account issues
- Using a ticketing system (osTicket)
- User onboarding and access control
- Documenting support tasks and resolutions

---
- Summary
This lab provides practical experience with real IT support workflows, including ticket creation, troubleshooting, resolution, and documentation. It reflects the type of tasks handled daily in a help desk or IT support role.
